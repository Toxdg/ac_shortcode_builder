# AC Shortcode builder

Plugin for administrator. It helps user to fill shortcodes.

## how it use

Upload to:

    wp-content/plugins

create a file in the **modules** folder by the schema in the file

    test_shortcode_1.php
or

    test_shortcode_2.php
